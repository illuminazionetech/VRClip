Come installarlo

  1. Scarica l'APK

    Tocca il pulsante di download qui sopra: il browser scaricherà il file adatto al tuo dispositivo (architettura arm64-v8a, quella usata dalla quasi totalità degli smartphone recenti).

  2. Consenti l'installazione da questa origine

    Se compare un avviso, vai su Impostazioni → App → Accesso speciale → Installa app sconosciute, seleziona il browser usato per scaricare il file e attiva "Consenti da questa origine".

  3. Apri il file scaricato

    Apri il file .apk dalle notifiche o dalla cartella Download e tocca "Installa". Al termine, apri VRClip e incolla il link del video che vuoi scaricare.

Non sei sicuro di quale file scegliere? L'APK arm64-v8a proposto dal pulsante va bene per praticamente ogni telefono e tablet Android prodotto negli ultimi anni.

  1. Attiva la modalità sviluppatore sul visore

    Nell'app Meta Horizon sul telefono, vai su Menu → Dispositivi, seleziona il tuo visore, apri Impostazioni per sviluppatori e attiva la modalità sviluppatore (richiede un account sviluppatore Meta gratuito).

  2. Scarica l'APK direttamente nel visore

    Indossa il visore, apri il browser di Meta Quest e vai su questa stessa pagina: verrà riconosciuta automaticamente e potrai toccare il pulsante di download qui sopra per scaricare l'APK giusto.

  3. Installa il file scaricato

    Apri l'app "File" nella libreria del visore (sezione "Sorgenti sconosciute" se necessario), vai nella cartella Download e tocca il file .apk per installarlo. Se il tuo File Manager non supporta l'installazione, usa SideQuest da PC per installarlo via cavo o Wi-Fi.

  4. Avvia VRClip

    Trovi l'app nella libreria, sezione "Origini sconosciute" / "App su questo dispositivo". VRClip riconosce automaticamente di girare su Quest e attiva subito il player immersivo.

In alternativa, puoi scaricare l'APK su un telefono o PC e installarlo via adb install nomefile.apk o con SideQuest, senza dover riscaricare il file nel visore.

Cosa fa VRClip

Download universale

Centinaia di siti supportati tramite yt-dlp, con aria2c integrato per il massimo della velocità.

Player immersivo

Riconoscimento automatico di flat, 360° mono/3D, 180° e stereoscopico side-by-side/top-bottom.

Nativo su Meta Quest

Scena spaziale reale tramite Meta Spatial SDK: pannello video posizionato nello spazio, stereo corretto per occhio.

Metadati automatici

Copertine, sottotitoli e tag incorporati automaticamente nei file scaricati.

Privacy prima di tutto

Nessun tracciamento, nessuna pubblicità, nessun account. I download avvengono localmente sul dispositivo.

Aggiornamenti costanti

App open source, in continuo sviluppo: aggiornamento automatico integrato direttamente dal repository GitHub.
