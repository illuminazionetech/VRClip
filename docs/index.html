How to install

  1. Download the APK

    Tap the download button above: your browser will fetch the right file for your device (the arm64-v8a architecture, used by nearly every recent phone).

  2. Allow installs from this source

    If a warning appears, go to Settings, Apps, Special access, Install unknown apps, select the browser you used for the download, and enable "Allow from this source".

  3. Open the downloaded file

    Open the .apk file from your notifications or the Download folder and tap "Install". When it finishes, open VRClip and paste the link of the video you want to download.

Not sure which file to pick? The arm64-v8a APK offered by the button works on practically every Android phone and tablet made in recent years.

  1. Enable developer mode on the headset

    In the Meta Horizon app on your phone, go to Menu, Devices, select your headset, open Developer settings, and enable developer mode (requires a free Meta developer account).

  2. Download the APK directly in the headset

    Put on the headset, open the Meta Quest browser, and visit this same page: it is detected automatically, and the download button above will fetch the right APK.

  3. Install the downloaded file

    Open the "Files" app in the headset library (under "Unknown sources" if needed), go to the Download folder, and tap the .apk file to install it. If your file manager cannot install it, use SideQuest from a PC over cable or Wi-Fi.

  4. Launch VRClip

    Find the app in your library under "Unknown sources" / "Apps on this device". VRClip detects that it is running on Quest and enables the immersive player right away.

Alternatively, download the APK on a phone or PC and install it with adb install filename.apk or SideQuest, without re-downloading it in the headset.

What VRClip does

Universal downloads

Hundreds of sites supported through yt-dlp, with bundled aria2c for maximum speed.

Immersive player

Automatic detection of flat, 360 mono/3D, 180, and stereoscopic side-by-side or top-bottom content.

Native on Meta Quest

A real spatial scene through the Meta Spatial SDK: a video panel placed in space with correct per-eye stereo.

Automatic metadata

Cover art, subtitles, and tags embedded automatically into downloaded files.

Privacy first

No tracking, no ads, no accounts. Downloads happen locally on your device.

Always up to date

Open source and in active development, with built-in automatic updates straight from the GitHub repository.
