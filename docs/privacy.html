Privacy policy

Last updated: July 18, 2026

VRClip does not collect, store, or transmit any personal data to VRClip's own servers. VRClip has no backend servers.

Network access

Network access is limited to:

  • Fetching media the user explicitly requests from a URL they provide, sent directly to the source site.
  • An optional check for yt-dlp engine updates, served from GitHub.
  • An optional check for app updates, served from GitHub Releases.

Analytics and tracking

No analytics, advertising, or tracking SDKs are included. No account or sign-in exists.

Data storage and deletion

All downloaded files and app preferences are stored locally on the device. They can be deleted at any time by removing the files, clearing the app's data, or uninstalling the app. There is no server-side account or data to delete.

Permissions

The storage permission is used solely to save downloads to the device's public Download folder. The notification permission is used solely to show download progress and completion. The install-packages permission is used solely to install app updates that the user explicitly confirms.

Contact

For questions or reports, open an issue at github.com/illuminazionetech/VRClip/issues.
